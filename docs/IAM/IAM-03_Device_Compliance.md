# 🔐 IAM-03: Device Compliance (Intune Integration)

## 📌 Objective
Ensure that only compliant and secure devices can access organizational resources.

---

## 🧠 Description
Device compliance policies in Microsoft Intune enforce security requirements on devices before granting access.

Conditional Access integrates with Intune to allow only:
- Compliant devices
- Managed devices

---

## ⚙️ Configuration Steps

### 🔹 Step 1: Navigate
- Microsoft Intune Admin Center
- Go to: Devices → Compliance policies
