## IAM-06 – Access Review

#### Ziel
Regelmäßige Überprüfung von Gruppenmitgliedschaften zur Sicherstellung des Least-Privilege-Prinzips.

### Umsetzung im Lab
- Access Review für SG-DEPT-IT erstellt
- Reviewer definiert
- Automatische Entfernung bei Nichtreaktion aktiviert
- Zugriff eines nicht mehr berechtigten Benutzers entfernt

### Ergebnis
Unnötige Berechtigungen wurden identifiziert und entzogen.

### Sicherheitsnutzen
- Reduzierung von Insider-Risiken
- Minimierung von Privilege Creep
- Umsetzung von ISO 27001 A.9
