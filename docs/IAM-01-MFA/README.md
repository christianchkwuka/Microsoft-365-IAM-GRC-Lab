# 🔐 IAM-01: Multi-Factor Authentication (MFA)

## Ziel
Erhöhung der Sicherheit durch zusätzliche Authentifizierung.

## Umsetzung
- MFA für Benutzer aktiviert
- Verwendung von Microsoft Authenticator

## Ergebnis
- Schutz vor unbefugtem Zugriff
- Erfüllung von Zero Trust Anforderungen
